Mac Apps

A list of third-party Mac software you may not know about but should.

  • User Experience
  • Utility
  • System
  • Security
TotalFinder
Do you wish your Mac had Cut & Paste like your old Windows computer did? Wish Finder had tabs like your internet browser does so you don't have 5 open Finder windows cluttering up your screen? Need to see your system files (you know, the ones that start with a . and are usually hidden unless you know how to use the command line or downloaded some software that lets you display them in Finder)?
TotalFinder has all of these and many other useful features. Not only that, but TotalFinder stays out of your way by always hiding its window until you decide you want to see it using a simple keyboard shortcut. Definitely one of my favorite third party applications.
Alfred
Alfred is like a Swiss-army knife for OS X and a keyboard-shortcut enthusiast's dream application. Its free version is fairly useful, but its paid "Powerpack" version is where it really shines. File System Navigation, built in Clipboard History, and being able to search one of many popular sites like Amazon, eBay, or Facebook with just a keyboard shortcut are some of the many features that make Alfred great.
KeyRemap4MacBook
This app allows you to customize your keyboard layout in a staggering amount of ways. If you only use modifier keys (Shift, Ctrl, Option, Command) on one side of the keyboard, you can edit the modifier keys on the other side to do other useful functions. You can also change what the F keys (top row) do if you don't like the default setup. Know Emacs or Vi shortcuts? You can now use them on your desktop!
This tool saved me after I bought a Happy Hacking Professional 2 Keyboard. IMO that keyboard's layout is amazing in every way except for one very important aspect - I didn't like using the Function key to enable the arrow key functions because Function key was too far away from the default hand positions; I was afraid I had just spent a ridiculous amount of money on a keyboard I won't even enjoy using. Luckily, this tool has features JUST for that keyboard! Instead of holding down the Function key, I can use the Shift key instead which is much closer to default hand positions.
BetterTouchTool
If you have a unibody Mac laptop with the multitouch trackpad or if you have a Magic Trackpad, this tool adds amazing new multitouch powers to your trackpad. You can assign almost any action you can think of to these gestures, allowing you accomplish tasks in fractions of the time. I won't even try listing some of them. Check it out for yourself on their website.
SizeUp
Allows you to reposition and resize any of your open windows in many useful ways using keyboard shortcuts. Saves you the time and annoyance of using your mouse cursor to do it. BetterTouchTool also allows you to do most of these functions but using the trackpad. Use both apps together for an awesome user experience!
Visor
Similar to TotalFinder but for the Terminal. Enables tabbed windows and hiding/showing the Terminal window using a simple keyboard shortcut.
