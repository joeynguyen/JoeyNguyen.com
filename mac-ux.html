Mac User Experience Apps

  • User Experience
  • Utility
  • System
  • Security
TotalFinder
  • Enables Cut & Paste for files on Mac OS just like on your old Windows computer
  • Tabbed browsing like your internet browser so you don't have multiple open Finder windows cluttering up your screen
  • Shows hidden system files without using the command line or installing a piece of software whose sole ability is showing hidden files in Finder
TotalFinder has all of these and many other interesting features in one package. Not only that, but TotalFinder has a feature that allows it to stay out of your way by always hiding its window until you decide you want to see it using a simple keyboard shortcut. Definitely one of my favorite third party applications.




Visor
Similar to TotalFinder but for the Terminal, Visor enables tabbed windows and hiding/showing of the Terminal window using a simple keyboard shortcut.




KeyRemap4MacBook
This app allows you to customize your keyboard layout in a staggering amount of ways. If you only use modifier keys (Shift, Ctrl, Option, Command) on one side of the keyboard, you can edit the modifier keys on the other side to perform other needed functions. You can also change what the F keys (top row) do if you don't like the default setup. Know Emacs or Vi? You can now use them outside of Terminal!
(Warning: this next paragraph will only be interesting to me and other ergonomic/mechanical keyboard enthusiasts. Read at your own peril.)
KeyRemap4MacBook saved my butt after I bought a Happy Hacking Professional 2 keyboard. In my opinion, this keyboard's layout is awesome for many reasons, but it fails in one very important aspect - reaching my pinky finger for the Function key to trigger the arrow keys functionality was uncomfortable because the key was slightly too far away from the default hand placement position. I was afraid I had just spent a ridiculous amount of money on a keyboard I wouldn't even enjoy using. Fortunately, I found KeyRemap4MacBook which has a set of features dedicated to this keyboard (other owners must have agreed with me). Instead of holding down the Function key, I can now hold down the right Shift key + IJKL instead (for arrows) which is much more comfortable.




BetterTouchTool
If you have a unibody Mac laptop with the multitouch trackpad, a Magic Trackpad, or Magic Mouse, BetterTouchTool can add amazing new multitouch powers to it. You can assign almost any action you can think of to these gestures, allowing you accomplish tasks in a fraction of the time. I'm not going to bother listing the types of gestures available; it's easier seeing it in action yourself in the video below (not mine).
SizeUp
SizeUp allows you to reposition and resize any of your open windows in many useful ways using keyboard shortcuts, which saves you the time and annoyance of using your mouse cursor to do it. BetterTouchTool also allows you to do most of these functions but only with performing trackpad gestures. Use both apps for an awesome user experience! Like BetterTouchTool, the quickest way to wrap your head around it is by watching a short video (also not mine).
