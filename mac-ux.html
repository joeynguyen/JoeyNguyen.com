Mac Apps

A list of third-party Mac software you may not know about but should.

  • User Experience
  • Utility
  • System
  • Security
TotalFinder
Do you wish your Mac had Cut & Paste like your old Windows computer did? Wish Finder had tabs like your internet browser does so you don't have 5 open Finder windows cluttering up your screen? Need to see your hidden system files without using the command line or downloading a piece of software solely for the purpose of viewing them in Finder?
TotalFinder has all of these and many other interesting features in one package. Not only that, but TotalFinder stays out of your way by always hiding its window until you decide you want to see it using a simple keyboard shortcut. Definitely one of my favorite third party applications.
Alfred
Alfred is like a Swiss-army knife for OS X and a keyboard-shortcut enthusiast's dream application. Its free version is fairly useful, but its paid "Powerpack" version is where it really shines. Command line style File System Navigation, built-in Clipboard History, and being able to search one of many popular sites like Amazon, eBay, and YouTube with just a keyboard shortcut are some of the many features that make Alfred great.
KeyRemap4MacBook
This app allows you to customize your keyboard layout in a staggering amount of ways. If you only use modifier keys (Shift, Ctrl, Option, Command) on one side of the keyboard, you can edit the modifier keys on the other side to perform other needed functions. You can also change what the F keys (top row) do if you don't like the default setup. Know Emacs or Vi? You can now use them outside of Terminal!
This tool saved my butt after I bought a Happy Hacking Professional 2 keyboard. In my opinion, this keyboard's layout is awesome for many reasons, but it fails in one very important aspect - reaching my pinky finger for the Function key to trigger the arrow keys functionality was uncomfortable because the key was slightly too far away from the default hand placement position. I was afraid I had just spent a ridiculous amount of money on a keyboard I wouldn't even enjoy using. Fortunately, I found KeyRemap4MacBook which has a set of features dedicated to this keyboard (other people must have agreed with me). Instead of holding down the Function key, I can now hold down the right Shift key + IJKL instead (for arrows) which is much more comfortable.
BetterTouchTool
If you have a unibody Mac laptop with the multitouch trackpad, a Magic Trackpad, or Magic Mouse, this tool adds amazing new multitouch powers to it. You can assign almost any action you can think of to these gestures, allowing you accomplish tasks in fractions of the time. I won't even try listing some of them; it's easier seeing it in action yourself below (not my video).
SizeUp
SizeUp allows you to reposition and resize any of your open windows in many useful ways using keyboard shortcuts, which saves you the time and annoyance of using your mouse cursor to do it. BetterTouchTool also allows you to do most of these functions but only with performing trackpad gestures. Use both apps for an awesome user experience! Like BetterTouchTool, the quickest way to wrap your head around it is by watching a short video (also not mine).
Visor
Similar to TotalFinder but for the Terminal. Enables tabbed windows and hiding/showing the Terminal window using a simple keyboard shortcut.
